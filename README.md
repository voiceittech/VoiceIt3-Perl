A Perl wrapper for VoiceIt's API 3.0 featuring Voice + Face Verification and Identification.

## Installation

```bash
git clone https://github.com/voiceittech/VoiceIt3-Perl.git
```

Then use in your script:
```perl
use lib "VoiceIt3-Perl";
use VoiceIt3;
```

## Getting Started

Sign up at [voiceit.io/pricing](https://voiceit.io/pricing) to get your API Key and Token, then log in to the [Dashboard](https://dashboard.voiceit.io) to manage your account.
